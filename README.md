# swing-jdbc
Java Swing simple application using IntelliJ Form. Application is using MySQL database using mysql connector.

# How to use this app
1. Open file database demo pp 20251.mwb in MySQL Workbench and forward engineer saved model to your MySQL Server.
2. Change URL, username, and password in MySQLUtil.
3. Add mysql-connector (latest version) to this project.
4. Run application from LoginForm.java. You can use robby.tan as username and 12345 as password
